# Brolly branding assets

Drop the following PNGs into this folder before running
`flutter pub run flutter_launcher_icons` / `flutter_native_splash`:

| File                              | Size           | Purpose                          |
| --------------------------------- | -------------- | -------------------------------- |
| `brolly_icon.png`                 | 1024 × 1024    | Main app icon                    |
| `brolly_icon_foreground.png`      | 1024 × 1024    | Android 12+ adaptive foreground  |
| `brolly_splash.png`               | 1152 × 1152    | Native splash mark               |

The Brolly mark is a simple umbrella silhouette with a radar sweep arc
emerging from the top — keep it monochrome so it works on light + dark.
Primary colour is `#1E5A8E`; splash background is also `#1E5A8E`
(dark-mode `#0B2E4E`).
